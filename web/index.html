Installation

To install IDK on macOS (ARM64), Linux (x64/ARM64), or WSL, run the following command in your terminal:

curl -sSL https://idk.victorlpgazolli.dev/install.sh | bashCopy

The installer automatically detects your architecture, sets up the environment, and removes macOS quarantine flags.
